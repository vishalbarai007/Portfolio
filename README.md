## 💻 Running the Project Locally

To explore the portfolio codebase on your local machine, follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/vishalbarai007/portfolio_2.git

```


2. Navigate into the project directory:
```bash
cd portfolio_2

```


3. Install the dependencies:
```bash
npm install

```


4. Start the development server:
```bash
npm run dev

```


5. Open your browser and navigate to `http://localhost:3000` to view the site.
